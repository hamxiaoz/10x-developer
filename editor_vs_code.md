# Editor: VS Code

## Hotkeys

Focus
- Toggle side pane: `CMD+B`
- Switch to file pane: `SHIFT+CMD+E`
- Switch to focus group 1/2: `CMD+1/2`

Search:
- Switch search history: `ALT+UP/DOWN`
